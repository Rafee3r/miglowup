# MiGlowUp 🌸

Landing de validación para una app fitness dirigida a mujeres chilenas 25-45.

**Estrategia:** simular que la app existe, cobrar $990 por trial de 7 días vía Mercado Pago, medir CAC con Facebook Ads. Si valida → construir la app real.

## Quick start

1. Abre `index.html` en el navegador para previsualizar.
2. Sigue `SETUP.md` para deploy + Mercado Pago + Meta Ads.

## Meta de validación

20-50 trials pagados en 14 días con CAC < $8.000 CLP.
